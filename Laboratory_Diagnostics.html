Test Preparation Guidelines

Proper preparation ensures accurate test results. Follow these guidelines for common tests.

Blood Tests

Fast for 8-12 hours before tests requiring fasting. Drink water unless instructed otherwise. Avoid strenuous exercise before testing.

Urine Tests

Collect first morning urine when possible. Clean the genital area before collection. Midstream collection is preferred for most tests.

Stool Tests

Use provided collection containers. Avoid contaminating with urine or toilet water. Refrigerate if not delivered within 2 hours.
